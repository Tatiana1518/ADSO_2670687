Concepto de GIT

Git es una de las herramientas más indispensables para el desarrollo de proyectos, aunque un sistema de control de versiones sirva justamente para controlar los estados por los que ha pasado un código, lo cierto es que la herramienta git facilita mucho el desarrollo de los proyectos en equipo, así como otras operaciones relacionadas con el flujo de trabajo de los desarrolladores.

PRICIPALES COMANDOS DE GIT

  1. GIT INIT:Este comando funciona para inicializar un proyecto con GIT
  2. GIT ADD:Este comando añade cambios de archivos en el directorio de ensayo
  3. GIT CONFIG --GLOBAL USER.NAME:Este comando establece valores de configuracion para tu usuarios para digitar el nombre del creador del repositorio.
  4. GIT CONFIG --GLOBAL USER.EMAIL:Este comando establexe valores de configuracion para los usuarios para digitar el email del creador del repositorio
  5. GIT CONFIG USER.NAME
  6. GIT CONFIG USER.EMAIL
  7. GIT STATUS:Este comando muestra el estado de los archivos, ennumera los archivos que no estan rastreados, modificados y almacenados.
  8. GIT COMMIT -M:Este comando toma todos los cambios escritos, crea un nuevo objeto de comfirmacion que apunta a el y establece una rama para que apunte a esa nueva confirmacion
  9. GIT LOG:Este comando muestra una lista de confirmacion en una rama que incluye los detalles correspondientes.
  10. GIT BRANCH:Este comando esta para listar las ramas existentes, incluyendo las ramas remotas, si se proporciona -a crea una nueva rama si se proporciona un nombre.
