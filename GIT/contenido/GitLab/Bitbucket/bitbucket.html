BITBUCKET

Visita la direccion de bitbucket en el siguiente enlace www.bitbucket.com

Bitbucket es un servicio de alojamiento de repositorios de control de versiones, que fue creado en 2008 y es propiedad de Atlassian. Esta solución de gestión de repositorios de Git está escrita en Python, y construida utilizando el marco web de Django Si bien puedes utilizar Bitbucket para alojar repositorios de código abierto, está orientado principalmente a empresas y negocios que desarrollan código privado y propietario. Otro beneficio único de Bitbucket es que ofrece una plataforma segura para tu código con certificación Soc 2 Tipo 2.
